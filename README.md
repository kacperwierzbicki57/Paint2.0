# Mini Paint  
**Lightweight Sketching and Drawing Application**

Mini Paint is a portfolio-oriented project that delivers quick and intuitive drawing capabilities without the complexity of full-scale graphic design software. It is designed for rapid sketching, simple illustrations, and visual annotations, making it ideal for idea exploration, documentation support, and lightweight creative tasks.

This project focuses on clean design, extensibility, and clear user interaction while demonstrating solid programming fundamentals.
